Less : Leaner Style Sheets

Less (which stands for Leaner Style Sheets) is a backwards-compatible language extension for CSS. This is the official documentation for Less, the language and Less.js, the JavaScript tool that converts your Less styles to CSS styles.

@base: #f938ad

Sets a base SCOPE variable and gives it a certain color.

box-shadow(@style, @c) when (iscolor(@c)) box-shadow: @style @c; -webkit-box-shadow: @style @c; -moz-box-shadow: @style @c; }

The LESS function that called when variable @C is color. The @style is the OFFSET for the RULE of BOX-SHADOW It also handles the Mozilla browser.

.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) .box-shadow(@style, rgba(0, 0, 0, @alpha)); }

A function that is used when @alpha is a number. For the RULL of BOX-SHADOW, it gets STYLE.
